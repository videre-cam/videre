# Videre                                                                                  
   
  Camera evaluation for Linux engineers.                                                    
                                                            
  **[get-videre.com](https://get-videre.com)** — download, documentation, and pricing.

  ---

  ## Bug reports

  Open an issue in this repository. Please use the bug report template and include:
  - Videre version (`videre --version`)
  - OS and kernel version
  - Camera model and connection type (USB, CSI)
  - Steps to reproduce
